Embed 3D Tour → любой сайт

Готовый блок кода для вставки 3D тура на существующий сайт.
Сайт не должен быть заранее подготовлен под тур — просто вставь код в нужное место HTML.

1
Задеплой 3D тур на Vercel

Сначала убедись, что сам тур (папка 3d-tour/) задеплоен и доступен по публичному HTTPS-адресу. После деплоя скопируй его URL.

bash terminal 
# В корневой папке проекта 3d-tour/
npx vercel --prod

# Получишь URL вида:
https://3d-tour-elite.vercel.app
2
Вставь этот блок кода в HTML сайта

Скопируй блок ниже и вставь в любое место страницы — между двумя секциями, после каталога, в страницу «О нас» и т.д. Никакой предварительной подготовки не нужно.

HTML вставить в index.html / любая страница 
<!-- ══════════════════════════════════════════════════════
     3D ТУР — вставьте этот блок в нужное место страницы
     Замените URL на ваш адрес из Vercel
     ══════════════════════════════════════════════════════ -->

<div style="width:100%; max-width:1200px; margin:60px auto;">

  <!-- Заголовок секции (необязательно) -->
  <h2 style="text-align:center; margin-bottom:24px;">
    Виртуальный тур по салону
  </h2>

  <!-- Сам iframe с туром -->
  <iframe
    src="https://ВАШ-ТУР.vercel.app"
    width="100%"
    height="600"
    frameborder="0"
    allowfullscreen
    allow="fullscreen; gyroscope; accelerometer"
    loading="lazy"
    sandbox="allow-scripts allow-same-origin allow-fullscreen"
    style="border-radius:12px; display:block;"
    title="Виртуальный 3D тур"
  ></iframe>

  <!-- Ссылка под туром (необязательно) -->
  <p style="text-align:center; margin-top:12px; font-size:0.85rem; opacity:0.6;">
    <a href="https://ВАШ-ТУР.vercel.app" target="_blank">
      Открыть в полном экране →
    </a>
  </p>

</div>

<!-- ══════════════════════════════════════════════════════ -->
3
Минимальный вариант (1 строка)

Если сайт уже стилизован и тебе нужно просто вставить тур без обёртки:

HTML минимум кода 
<iframe src="https://ВАШ-ТУР.vercel.app" width="100%" height="600" frameborder="0" allowfullscreen></iframe>
4
Куда вставить в существующем сайте?

Примеры — тур органично вписывается в любое место:

HTML site.html — пример вставки после секции каталога 
<!-- ... существующий код каталога ... -->
</section>

<!-- 3D ТУР (вставили между секциями) ─────────────────────────── -->
<section style="padding:60px 20px; background:#f5f5f5;">
  <div style="max-width:1200px; margin:0 auto;">
    <iframe
      src="https://ВАШ-ТУР.vercel.app"
      width="100%" height="600" frameborder="0"
      allowfullscreen loading="lazy"
      sandbox="allow-scripts allow-same-origin allow-fullscreen"
    ></iframe>
  </div>
</section>
<!-- ──────────────────────────────────────────────────────────── -->

<!-- ... следующая секция сайта ... -->
<section class="new-arrivals">
✅ Не нужен CSS на сайте ✅ Не нужен JS на сайте ✅ Нет конфликтов со стилями ✅ Работает на любом CMS (WordPress, Bitrix и т.д.) ✅ Тур обновляется автоматически
🎯

Итог: тур на сайте — это буквально одна строка кода

Сайт ничего не знает о туре. Тур ничего не знает о сайте. Они существуют независимо. Обновляешь тур на Vercel — он автоматически обновляется везде, где встроен iframe.
